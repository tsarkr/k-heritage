# K-Heritage

**국가유산채널(K-heritage.tv) 유튜브 채널의 댓글 분석**  
**Comment Analysis of the K-heritage.tv YouTube Channel**

---

## 📋 주요 기능
- **댓글 수집**
  - YouTube API를 활용한 댓글 데이터 수집
  *YouTube API 필요*
  - 대량 데이터를 효과적으로 처리

- **댓글 분석**
  - 자연어 처리(NLP) 기술을 활용한 텍스트 분석
  - 키워드 및 주제 추출

- **데이터 시각화**
  - 분석 결과를 그래프 및 차트로 시각화
  - 결과를 직관적으로 이해할 수 있는 대시보드 제공

---
